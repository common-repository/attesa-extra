Add extra features to Attesa WordPress theme

== Description ==

<strong>Attesa Extra</strong> add extra features to <a href="https://attesawp.com" target="_blank">Attesa Theme</a> like custom templates, widgets, elementor widgets and metaboxes to edit the structure of the website for each post/page. You can also <a href="https://attesawp.com/demos/" target="_blank">import demo content</a> with a click.<br />
Attesa Extra <strong>requires free <a href="https://attesawp.com" target="_blank">Attesa theme</a></strong> to be installed.

== Installation ==

1. Make sure you have Attesa Theme installed
2. Upload the folder `attesa-extra` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= I installed the plugin but it does not work =
Make sure you have <a href="https://attesawp.com" target="_blank">Attesa Theme</a> installed because Attesa Extra is a plugin designed to work exclusively with free Attesa theme
